# Input

## 요구 사항

- input 창에 이메일과 비밀번호를 입력할 수 있어야 합니다.

- (부가 요소) 이메일 형식에 맞을 경우 체크 표시 등으로 알려주어야 합니다.

- (부가 요소) 비밀번호 입력란 우측 눈 아이콘을 누르면 비밀번호가 가려지지 않고 노출되어야 합니다.
